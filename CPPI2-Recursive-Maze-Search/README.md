# Recursive Maze Search

Use recursion and backtracking to find a path through a small grid.

## Tasks

1. Read the starter code and identify the missing behavior.
2. Complete the TODOs without changing the public input/output shape.
3. Add or run at least two custom checks.
4. Compare your result to the reference solution after documenting your reasoning.

## Build

From the repository root:

```bash
cmake -S . -B build
cmake --build build
```

Each project also contains a standalone `starter/main.cpp` and `solution/main.cpp` file that can be compiled directly with a C++17 compiler.
